# GurlFind

google search using GurlFind. this tool find 1000+ URLs as per your query


# Requirements


• Python 3

• Requests

• BeautifulSoup

• google
## Termux Installation


```bash
git clone https://github.com/cyberxaman/Gurlfind.git
cd Gurlfind
chmod +x *
./setup.sh
```

## Linux Installation


```bash
git clone https://github.com/cyberxaman/Gurlfind.git
cd Gurlfind
chmod +x *
sudo ./setup.sh
```

## How To Use 

```bash
python3 Gurlfind.py
 ```

Enter your query or website name. The script will returns Urls.
